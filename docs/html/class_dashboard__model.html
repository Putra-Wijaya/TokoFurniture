Detailed Description

Class Dashboard_model

Kelas ini berfungsi untuk mengambil data terkait dengan dashboard admin, seperti total pendapatan, pelanggan, barang, penjualan, dan pesanan terbaru. Ini menggunakan model CodeIgniter untuk berinteraksi dengan database.
